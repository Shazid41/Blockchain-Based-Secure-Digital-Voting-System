# Fraud Detection

Fraud detection is rule-based. The system does not use machine learning.

Logged events include:
- repeated wrong login
- unapproved voter trying to vote
- suspended account trying to vote
- duplicate vote attempt
- wrong-region vote attempt
- vote before election start
- vote after election end
- invalid candidate
- blockchain mismatch
- missing vote block
- suspicious repeated requests

Admins can review and resolve fraud alerts from the Fraud Alerts page.
