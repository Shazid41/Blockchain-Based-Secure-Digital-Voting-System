# Final Project Report

## Project
Blockchain-Based Secure Digital Voting System

## Academic Context
Rabindra Maitree University, CSE 3208 - Web Engineering Lab.

## Completed Features
- Secure voter registration and email verification structure
- Role-based voter/admin routes
- Admin voter approval and management UI
- Election, candidate, and region management
- Eligibility preparation
- Secure vote casting RPC
- Anonymous ballot storage
- Blockchain-inspired vote blocks
- Receipt verification
- Aggregate-only results
- Rule-based fraud logs
- Audit logs
- GitHub Actions validation and deployment drafts

## Disclaimer
This is an academic prototype. It is not ready for national or government elections. Real systems require certified identity verification, legal compliance, independent security audits, penetration testing, formal cryptographic review, secure infrastructure, disaster recovery, and public oversight.
