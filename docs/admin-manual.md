# Admin Manual

1. Open the admin dashboard to review summary cards and pending work.
2. Manage voters from the voters page. Confirm before approving, rejecting, suspending, or restoring pending status.
3. Manage regions carefully. A region used by an election should not be deleted without safe handling.
4. Create elections with valid start and end times.
5. Add candidates and ensure each candidate belongs to an election.
6. Prepare voter eligibility before secure voting is enabled in Step 3.
7. Review aggregate results without exposing individual ballots.
8. Run blockchain audit verification for completed or active elections.
9. Review fraud alerts and mark resolved items only after investigation.
10. Do not store secrets in settings.
